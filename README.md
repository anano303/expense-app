# Expense Management App

рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃЮрЃарЃњрЃљрЃюрЃўрЃќрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃўрЃњрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃю CRUD рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃА, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА, рЃЊрЃљ рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА. Express.js рЃЊрЃљ React.js рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃЉрЃћрЃЦрЃћрЃюрЃЊрЃўрЃАрЃљ рЃЊрЃљ рЃцрЃарЃЮрЃюрЃбрЃћрЃюрЃЊрЃўрЃА рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.

## ­ЪЏа Features

РъА№ИЈ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ: рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃерЃћрЃДрЃЋрЃљрЃюрЃљ.
рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ: рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ.
рЃгрЃљрЃерЃџрЃљ: рЃљрЃарЃљрЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ.
РъА№ИЈ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃАрЃћрЃАрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ
JWT рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃў: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮ рЃерЃћрЃАрЃЋрЃџрЃљ рЃЊрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃўрЃА рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў.
рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ: рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃгрЃЮрЃарЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ.
РъА№ИЈ рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃърЃљрЃюрЃћрЃџрЃў
рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.
CRUD рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃўрЃарЃћрЃЉрЃљ.
РъА№ИЈ рЃърЃљрЃњрЃўрЃюрЃљрЃфрЃўрЃљ
рЃ«рЃљрЃарЃ»рЃћрЃЉрЃўрЃА рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.
